The React Compiler aims to eliminate manual memoization by understanding component semantics. That means less `useMemo` fatigue and fewer accidental performance regressions.

## What changes for teams

- Components can stay simple without losing performance.
- You can spend more time on user-facing improvements instead of performance micromanagement.

## How to prepare

Focus on idiomatic React patterns. Avoid mutating state, rely on pure components, and keep side effects predictable.
